# Freedom Countdown

A beautiful countdown website showing the time until your military service ends.

## Release Date

**September 12, 2026** (11 months from January 12, 2026)

## Features

- Real-time countdown with days, hours, minutes, and seconds
- Modern, responsive design with dark theme
- Smooth animations and hover effects
- Mobile-friendly layout

## How to Use

1. Open `index.html` in your browser
2. The countdown will automatically calculate the time until September 12, 2026

## Customization

To change the release date, edit the `releaseDate` variable in `script.js`:

```javascript
const releaseDate = new Date('YOUR_DATE_HERE').getTime();
```
